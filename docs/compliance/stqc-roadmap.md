# STQC Certification Roadmap

## 1. Objective
Achieve STQC (Standardisation Testing and Quality Certification) for Video Surveillance Systems (VSS).

## 2. Timeline (12 Months)

### Phase A: Internal Readiness (Months 0-3)
- **Goal:** Documentation & Process Stability.
- **Tasks:**
    - Complete Phase 0.5 Security Baseline.
    - Finalize User Manuals & Admin Guides.
    - Generate initial Functional Test Reports (Internal).

### Phase B: Pre-Assessment (Months 3-6)
- **Goal:** Gap Analysis.
- **Tasks:**
    - Engage 3rd Party Lab for "Dry Run" penetration test.
    - Fix Critical/High vulnerabilities.
    - Audit Evidence Repository structure.

### Phase C: Submission & Certification (Months 6-12)
- **Goal:** Official Stamp.
- **Tasks:**
    - Submit application to STQC.
    - On-site Witness Testing (Functional).
    - Vulnerability Assessment (Security).
    - Address Non-Conformities (NCs).
    - Receive Certificate.

## 3. Roles
- **Owner:** VMS Product Manager.
- **Reviewer:** Principal Architect / Security Lead.
- **Approver:** VP of Engineering.
